JamiDB_csv_date_time.exe creates csv of "history.db" with extra column named "date_time".
"date_time" contains converted timestamp data in readable date-time format.

Export or copy the history.db file of Jami to 
the same directory as JamiDB_csv_date_time.exe.

thereafter

Either double click exe file 

or

Open CMD and navigate to directory which contain exe file. Enter JamiDB_csv_date_time.exe  

Both options will create a file named 'interactions.csv' in the same directory.

Open 'interactions.csv' in Timeline Explorer (https://ericzimmerman.github.io/#!index.md).

All info will NOT be automatically readable in Microsoft Excel.
